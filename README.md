# to-do-List
A simple to-do List program entirely made in HTML, Javascript, CSS and Bootstrap, you can add some tasks and it will use localStorage to save them so you can use it later and can remove some task that has been done or that won't be made anymore.

#futue updates
In future will be features to delegate tasks or share with other people, a login system, a calendar integration to allow users schedule their tasks, there will be a comment system as well that will allow users to comment their tasks and add subtasks.
